# 🎉 EventMate - AI Planner and Scheduler

## 📌 Overview
EventMate is an intelligent event management platform designed to simplify the process of planning, booking, and organizing events. From venue booking to scheduling and coordination, EventMate leverages smart automation and an intuitive interface to enhance the experience for both users and event organizers.

Built with a React-based frontend and a Spring Boot backend, EventMate ensures smooth performance, secure data management, and a user-friendly experience.

---

## 🎯 Features

### ✅ User & Admin Authentication
- **User Login / Register:** Provides secure access for event planners and users.
- **Admin Login:** Enables administrators to manage halls, bookings, and user activities securely.

### ✅ Smart Hall Booking System
- Users can view available halls and book them easily for their events.
- Hall owners and admins can monitor and manage bookings efficiently.

### ✅ AI-Based Event Scheduling
- Automatically suggests optimal event dates and times based on user preferences and availability.

### ✅ User & Admin Dashboards
- **Users:** Track bookings and view event status.
- **Admins:** Manage halls, users, and overall system activities.

### ✅ Contact & Feedback System
- Allows users to contact support and share feedback about their event experiences.

### ✅ Attractive & Responsive UI
- Designed using React and CSS to provide a modern, responsive, and user-friendly interface.

---

## 🔧 Technologies Used

- **Frontend:** React.js, CSS  
- **Backend:** Java, Spring Boot  
- **Database:** MySQL  
- **AI Integration:** OpenAI API
